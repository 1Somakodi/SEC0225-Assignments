# SEC0225-Assignments
Welcome to the SEC0225 Assignments Repository.
This repository contains programming assignments, exercises, and practice tasks for students enrolled in the SEC0225 Software Engineering Class.

# Purpose
The goal of this repository is to:
1. Provide students with practical coding exercises.
2. Help students apply concepts learned during class.
3. Build problem-solving and programming skills through hands-on practice.
4. Track assignment submissions using Git and GitHub.

# Instructions
1. Review the assignment provided in this repository.
2. Create your solution using Python.
3. Test your code to ensure it works correctly.
4. Commit and push your changes to your own GitHub repository.
5. Submit your work by sharing the link to your assignment repository with your instructor.

# Submission Guidelines
Your assignment repository should:
1. Contain all source code files.
2. Include clear and meaningful commit messages.
3. Have a README file explaining your solution (where applicable).
4. Be accessible to the instructor for review. If private repo add info.somakodi@gmail.com as a contributer

# Example Submission
After completing your assignment, send a GitHub repository link similar to: https://github.com/yourusername/sec0225-assignment-01

# Academic Integrity
Students are encouraged to discuss concepts and learn together. However, all submitted code should be your own work. Copying another student's solution and presenting it as your own is not permitted.

# Need Help?
If you encounter any difficulties while working on an assignment:
1. Review the class notes and learning materials.
2. Revisit examples covered during lessons.
3. Ask questions during class or through the designated communication channels.
